Border Width

As utilidades de border-width do Tailwind controlam a espessura das bordas. As classes incluem border (padrão 1px), border-0, border-2, border-4, border-8 e também variantes por lado (border-t-*, border-r-*, border-b-*, border-l-*), e por eixo (border-x-*, border-y-*). :contentReference[oaicite:0]{index=0}

Valores padrão

Exemplos visuais: cada caixa mostra a mesma cor de fundo e uma borda com a espessura indicada.

border (1px)
border-0 (sem borda)
border-2
border-4
border-8
border-[12px] (arbitrário)

<div class="border-0">sem borda</div>

<div class="border-4">border-4</div>
<div class="border-8">border-8</div>

Eixos (border-x / border-y)

Aplicam a largura nos eixos inline ou block (ambas as bordas do eixo).

border-x (1px nas laterais)
border-x-4 (4px nas laterais)
border-y-8 (8px top/bottom)
<div class="border-x">...</div>
<div class="border-x-4">...</div>
<div class="border-y-8">...</div>

Start / End (direção lógica)

Use border-s-* / border-e-* para suportar LTR / RTL (inline-start / inline-end).

border-s-4 (start)
border-e-4 (end)
dir="rtl" (start vira direita)
<div class="border-s-4">...</div>
<div class="border-e-4">...</div>
<div dir="rtl" class="border-s-4">...</div>

Custom properties e valores arbitrários

É possível usar border-[] ou variáveis CSS: border-[var(--my-border)]. Use isso para casos em que os tokens do tema não cobrem a necessidade. :contentReference[oaicite:2]{index=2}

border-[5px]
border from var (--my-border)
border-[0.4rem]
:root { --my-border: 6px; }

<div class="border-[5px]">...</div>
<div style="--my-border:6px;" class="border-[var(--my-border)]">...</div>
<div class="border-[0.4rem]">...</div>

Estados e responsividade

Combine com pseudo-classes e breakpoints: hover:border-4, sm:border-2 md:border-4, etc.

hover:border-4
sm:border-4 md:border-8
hover:add border
<div class="hover:border-4">...</div>
<div class="sm:border-2 md:border-4">...</div>
<div class="border-0 hover:border">...</div>

Dica: prefira os tokens do tema (ex.: border-2, border-4) para consistência visual; use valores arbitrários ou variáveis CSS quando precisar de um ajuste fino. Para mais detalhes veja a documentação oficial. :contentReference[oaicite:3]{index=3}

Border Style

Utilitários para controlar o border-style no Tailwind. Classes principais: border-solid, border-dashed, border-dotted, border-double, border-hidden e border-none. Use divide-* para controlar o estilo entre filhos. :contentReference[oaicite:1]{index=1}

Remover borda

Use border-none para remover uma borda aplicada anteriormente (útil em breakpoints).

border-none (sem borda)
border-solid (com borda)
<button class="border-none">Sem borda</button>
<div class="border-2 md:border-none">Com borda em mobile, sem borda em md+</div>

Responsivo e estados

Prefixe com breakpoints (sm:, md:) ou com estados (hover:, focus:) para aplicar condicionalmente.

hover:border-dashed
md:border-dotted
hover:add style
<div class="border-solid hover:border-dashed">...</div>
<div class="border-solid md:border-dotted">...</div>
<div class="border-none hover:border-solid">...</div>

Dica: combine border-style com largura e cor para obter o efeito visual desejado; prefira tokens do tema para manter consistência, e use divide-* quando precisar aplicar estilo entre filhos. Para referência completa consulte a documentação oficial. :contentReference[oaicite:3]{index=3}
