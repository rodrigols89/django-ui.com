Padding

O padding é o espaçamento interno de um elemento, ou seja, a distância entre o conteúdo e sua borda.

Sem padding

Texto dentro da caixa (sem padding)

Com padding

Texto dentro da caixa (com p-6)

No TailwindCSS, o padding é controlado pelas classes p-*, px-*, py-*, pt-*, pr-*, pb-* e pl-*. Cada classe aplica espaçamento interno (padding) em direções diferentes.

1. Padding em todos os lados (p-*)

Este elemento tem p-0

Este elemento tem p-1

Este elemento tem p-2

Este elemento tem p-3

Este elemento tem p-4


<div class="p-4 bg-blue-200">
    Padding aplicado em todos os lados
</div>

2. Padding no eixo X (px-*)

Este elemento tem px-8

<div class="px-8 bg-green-200">
    Padding aplicado apenas na esquerda e direita
</div>

3. Padding no eixo Y (py-*)

Este elemento tem py-6

<div class="py-6 bg-purple-200">
    Padding aplicado apenas no topo e embaixo
</div>

4. Padding individual (top, right, bottom, left)

pt-8 pr-4 pb-2 pl-10 

<div class="pt-8 pr-4 pb-2 pl-10 bg-yellow-200">
    Padding diferente em cada lado
</div>

5. Escala de valores

O TailwindCSS usa uma escala numérica para padding, geralmente múltiplos de 0.25rem (4px). Por exemplo:

  • p-0 → 0px
  • p-1 → 0.25rem (4px)
  • p-2 → 0.5rem (8px)
  • p-4 → 1rem (16px)
  • p-8 → 2rem (32px)

Margin

A margin é o espaçamento externo de um elemento, ou seja, a distância entre a borda do elemento e os elementos vizinhos.

Sem margin

Caixa sem margem
Outra caixa sem margem

Com margin

Caixa com m-8

No TailwindCSS, a margin é controlada pelas classes m-*, mx-*, my-*, mt-*, mr-*, mb-* e ml-*. Cada classe aplica espaçamento externo (margin) em direções diferentes.

1. Margin em todos os lados (m-*)

4. Margin individual (top, right, bottom, left)

mt-4 mr-8 mb-2 ml-6

5. Escala de valores

O TailwindCSS usa a mesma escala numérica para margin, geralmente múltiplos de 0.25rem (4px). Por exemplo:

  • m-0 → 0px
  • m-1 → 0.25rem (4px)
  • m-2 → 0.5rem (8px)
  • m-4 → 1rem (16px)
  • m-8 → 2rem (32px)
